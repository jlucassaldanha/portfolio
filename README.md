# Portfólio Pessoal

Repositório contendo o código-fonte do meu portfólio. O objetivo desta página é me apresentar de forma simples e objetiva, meus contatos e uma seleção dos meus principais projetos.

🌍 **[Acesse o portfólio online clicando aqui](https://joaolucasportfolio.vercel.app)**

## 🛠️ Tecnologias e Práticas

O desenvolvimento das minhas aplicações (e deste portfólio) apoia-se em:

- **Frontend:** TypeScript, React e Next.js.
- **Backend:** C# (.NET), Python (FastAPI), PostgreSQL e Entity Framework Core.
- **Testes:** Cultura de testes unitários (utilizando ferramentas como Vitest).
- **Engenharia de Software:** Aplicação rigorosa de princípios **SOLID** e **Clean Code**. Estruturação de regras de negócio isoladas através de **Casos de Uso (Use Cases)**, evitando lógicas acopladas e garantindo sistemas escaláveis.

## ⚙️ Como Executar Localmente

Caso queira clonar este repositório e rodar o projeto localmente, siga os passos:

```bash
# Clone o repositório
git clone https://github.com/jlucassaldanha/portfolio.git

# Acesse o diretório
cd portfolio

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev

```
